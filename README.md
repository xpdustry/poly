## Description

Redirect players.

## Installation

The plugin requires:

- Java 25 or above.
- Mindustry v157 or above.

## Building

- `./gradlew :shadowJar` to compile the plugin into a usable jar (will be located at `build/libs/poly.jar`).
- `./gradlew :runMindustryServer` to run the plugin in a local Mindustry server.
- `./gradlew :runMindustryDesktop` to start a local Mindustry client that will let you test the plugin.
